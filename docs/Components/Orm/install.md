# ORM
Easyswoole提供的一个全新协程安全的ORM封装。
# 安装

依赖关系

- swoole `>= 4.4.8`
- Easyswoole  `>=3.3.2` 
- mysqli > `2.x`

```php
composer require easyswoole/orm
```
